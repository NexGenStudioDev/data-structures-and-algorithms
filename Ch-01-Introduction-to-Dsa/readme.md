# Type of Data & Data Structure

## What is Data?

Data refers to any information that is processed, stored, or handled by a computer.  
It can be in the form of numbers, text, symbols, images, audio, or any other type of digital information.

### 1) Raw Data
- Raw data is unprocessed information that has not yet been organized or structured for analysis.
- It is in its original form.
- **Example:** Numbers or text entered by a user before processing.

### 2) Processed Data
- Processed data is data that has been organized, cleaned, or transformed into a useful format.
- It is often represented in tables, graphs, or reports.
- **Example:** A report generated from raw input data.

---

## What is Data Structure?

A **data structure** is a way of storing and organizing data so that it can be accessed and manipulated efficiently.

In simple words, data structures help us manage data in a smart way so programs can run faster and use memory effectively.

> A data structure is **not** a programming language like C, C++, Java, or JavaScript. It is a concept used inside programming to organize data efficiently.

---

## Why Do We Use DSA?

1. DSA helps you solve problems more efficiently.  
2. Choosing the right data structure can make your program run faster and use less memory.  
3. It helps in finding optimized solutions, especially when dealing with large amounts of data.  
4. DSA breaks big and complex problems into smaller parts.

## Types of Data Structures

### 1) Primitive Data Structure
Primitive data structures are the basic data types provided by programming languages.  
They store a single value of one specific type.

**Examples:** Integer, Float, Character, String, Boolean.

### 2) Non-Primitive Data Structure
Non-primitive data structures are derived from primitive data types.  
They can store collections of similar or different types of data.

**Examples:** Array, Linked List, Stack, Queue, Tree, Graph.

---

## Linear Data Structure

Linear data structures store data in a sequential order.  
Each element is connected to the previous and the next element in a straight-line arrangement.

### 1) Array
- An array is a collection of elements stored in contiguous memory locations.

### 2) Linked List
- A linked list is a sequence of elements where each element points to the next element.

### 3) Stack
- A stack is a collection of elements that follows the **LIFO** principle: Last In, First Out.

### 4) Queue
- A queue is a collection of elements that follows the **FIFO** principle: First In, First Out.

---

## Non-Linear Data Structure

Non-linear data structures are those in which elements are not arranged sequentially.  
In these structures, one element can be connected to multiple other elements.

### 1) Tree
- A tree is a hierarchical data structure used to represent data in the form of parent-child relationships.

### 2) Graph
- A graph is a collection of nodes connected by edges.  
- Graphs are used to represent networks such as social networks, maps, and routing systems.
